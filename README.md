## サービス概要
### 声ヨガ
前向きな文章を音読することで前向きになれるサービス。

音声を感情判定をしてポジティブさによって色で判定したい。

### メインのターゲットユーザー
アファメーションに興味がある人。
自分で自分を励ましせるようになりたい。

### ユーザーが抱える課題
他人がアファメーションしてくれるコンテンツはあるけれど、自分でするのを促すようなコンテンツがないような気がする。
一人でやっても何か手応えがない。

### 解決方法
前向きな文章を用意して読んでもらう。
感情判定をして何かしらの反応を返すことで楽しくなるようなサービスを作る。

### 主な機能

- 感情判定
- 色
- 音声投稿
